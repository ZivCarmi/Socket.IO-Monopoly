# Socket.IO-Monopoly

A regular standart Monopoly Online game based on Socket-IO.

The game includes features such as:

- Lobby & game rooms
- Player turns
- Rolling dices & moving the player on the board
- Properties purchase & house buildings
- Chance & Surprise cards

# Getting Started

To run the app:

```
cd client;

npm install;

npm run dev
```

To run the server:

```
cd server;

npm install;

npm run dev
```
